## Make Repository Private

To make your repository private:

1. Go to your repository: https://github.com/andrea-catherine/legible-community-fb
2. Click on **Settings** (top navigation bar)
3. Scroll down to the **Danger Zone** section (at the bottom)
4. Click **Change visibility**
5. Select **Make private**
6. Type the repository name to confirm: `andrea-catherine/legible-community-fb`
7. Click **I understand, change repository visibility**

That's it! Your repository will now be private.
